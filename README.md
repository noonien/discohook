# discohook
[![GoDoc](https://godoc.org/github.com/noonien/discohook?status.png)](https://godoc.org/github.com/noonien/discohook)

Go library for sending messages on Discord webhooks

## Install
```shell
go get github.com/noonien/discohook
```
